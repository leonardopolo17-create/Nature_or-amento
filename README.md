# Nature — Gerador de Orçamentos
## Tutorial de instalação no Railway (10 minutos)

---

### PASSO 1 — Subir o código no GitHub

1. Acesse **github.com** e clique em **"New repository"** (botão verde)
2. Nome do repositório: `nature-orcamentos`
3. Deixe como **Public**
4. Clique em **"Create repository"**
5. Na página do repositório, clique em **"uploading an existing file"**
6. Arraste TODOS os arquivos desta pasta (app.py, requirements.txt, Procfile, e a pasta assets/)
7. Clique em **"Commit changes"**

---

### PASSO 2 — Conectar ao Railway

1. Acesse **railway.app** e faça login
2. Clique em **"New Project"**
3. Escolha **"Deploy from GitHub repo"**
4. Selecione o repositório `nature-orcamentos`
5. Railway detecta automaticamente que é Python e faz o deploy

---

### PASSO 3 — Gerar o link público

1. Após o deploy (1-2 minutos), clique no projeto
2. Vá em **"Settings"** → **"Networking"**
3. Clique em **"Generate Domain"**
4. Seu link será algo como: `nature-orcamentos.up.railway.app`

---

### PASSO 4 — Compartilhar com os vendedores

Mande o link no WhatsApp da equipe. Pronto!
O sistema funciona no celular e no computador.

---

## Como usar o sistema

1. Abra o link no celular ou computador
2. Preencha os dados do cliente
3. Selecione o produto e versão
4. Marque os adicionais inclusos
5. Digite o valor final do combo
6. Ative "Incluir fotos" e suba as fotos do modelo (opcional)
7. Clique em **"Gerar PDF do Orçamento"**
8. O PDF baixa automaticamente — pronto para enviar ao cliente!
